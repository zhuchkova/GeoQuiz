## Requirements
Install following packages
- wikipedia
- python-dotenv
- openai
- SPARQLWrapper

## Setup
Create and activate a virtual environment, then install the required packages:
```python3 -m venv .venv
source .venv/bin/activate
pip install wikipedia SPARQLWrapper openai python-dotenv
```

## To run Quiz
Run the main.py file.

`python main.py`
